# **App Name**: AutoPilot Careers

## Core Features:

- User Profile Management: Users create and manage their profiles with details like name, email, phone, location preferences, availability, experience, keywords, resume PDF(s), cover letter templates, and skill tags.
- Job Search Automation: Continuously searches job boards for new postings matching user criteria.
- Automated Job Application: Automatically applies to jobs using the user's uploaded resume and prefilled details via browser extension.
- Browser Extension Integration: A browser extension (Chrome first) that autofills forms, clicks submit buttons, and attaches resumes with explicit user authorization.
- AI-Powered Cover Letter Tailoring: Generates tailored cover letters for each job application using OpenAI tool or local templating, optionally selecting the best resume variant.
- Targeted Candidate Profile Optimizer: Analyzes job descriptions and suggests three small changes to the resume to improve match score using generative AI tool.
- Smart Rate-Limited Application Mode: Spreads applications across time to avoid spammy behavior and stay within safe platform thresholds.

## Style Guidelines:

- Primary color: Sky blue (#75baff), reflecting trustworthiness and technological prowess, important for a professional tool.
- Background color: Light gray (#f0f2f5), provides a clean and professional backdrop, promoting readability.
- Accent color: Teal (#45b8ac), for interactive elements, providing a sense of innovation and focus, and creating a balanced contrast.
- Body and headline font: 'Inter' sans-serif for a modern, clean, and highly readable interface.
- Minimalist line icons representing job categories, application status, and user profile settings.
- Clean and structured layout with clear hierarchy, emphasizing key actions and information for optimal user flow.
- Subtle transitions and loading animations to provide feedback and enhance user experience without being intrusive.
